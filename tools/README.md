# xript Tools

Tooling for the xript ecosystem.

## Available Tools

- **manifest-validator/** (`@xript/manifest-validator`) — validate xript manifests against the spec schema
- **typegen/** (`@xript/typegen`) — generate TypeScript definitions from manifests
- **docgen/** (`@xript/docgen`) — generate markdown documentation from manifests

## Planned

- **playground/** — interactive playground for testing scripts against manifests (see live demo issues [#47](https://github.com/nekoyoubi/xript/issues/47), [#48](https://github.com/nekoyoubi/xript/issues/48), [#49](https://github.com/nekoyoubi/xript/issues/49))
